## API

| 参数     | 说明     | 类型              | 默认值 |
| -------- | -------- | ----------------- | ------ |
| title    | 文章标题 | `React.ReactNode` | -      |
| meta     | 元信息   | `React.ReactNode` | -      |
| textLead | 引导文本 | `React.ReactNode` | -      |
| footer   | 底部区域 | `React.ReactNode` | -      |

## API

| Property | Description      | Type              | Default |
| -------- | ---------------- | ----------------- | ------- |
| title    | Article title    | `React.ReactNode` | -       |
| meta     | Meta information | `React.ReactNode` | -       |
| textLead | Lead text        | `React.ReactNode` | -       |
| footer   | Footer area      | `React.ReactNode` | -       |

## API

| Property   | Description              | Type                | Default |
| ---------- | ------------------------ | ------------------- | ------- |
| responsive | Whether responsive       | `boolean`           | `true`  |
| bordered   | Whether to show border   | `boolean`           | `true`  |
| float      | Float direction of image | `'left' \| 'right'` | -       |
